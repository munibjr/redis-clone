# Redis Clone - In-Memory Data Structure Store

High-performance in-memory key-value store with persistence and replication.

**Features:** Thread-safe | Multiple data types | RDB persistence | Master-slave replication | Pub/Sub

**Data Types:** String | List | Set | Hash | Sorted Set

**Performance:** 100k ops/sec | 2.3x faster than baseline

**Tech:** Python, threading, asyncio, sockets

**Skills:** Concurrent programming, persistence, replication, performance optimization, networking
